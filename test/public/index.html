뜻겹
우리말샘(OpenDict) 기반 “설명 가능한” 추론 게임

입력

우리말샘에 등록된 단어만 인정됩니다.

점수 규칙 (합리적 추론)

  • 의미 점수: 내 단어 뜻풀이 vs 정답 뜻풀이에서 핵심 키워드 겹침으로 계산 (가중치 Jaccard).
  • 공통 키워드: 두 뜻풀이에서 겹친 단어를 그대로 보여줘요.
  • “왜 점수가 높았는지”가 보여서 논리적으로 좁히기가 쉬워요.
※ 동적 버전: 단어/뜻풀이를 매번 우리말샘 API에서 가져옵니다.

시도 기록

# 입력 의미점수 공통키워드 추가단서
진행 상황은 로컬 저장(LocalStorage)에 저장됩니다. · 백엔드: Cloudflare Pages Functions(또는 Workers)에서 우리말샘 API를 프록시합니다.

게임 방법

  1. 단어를 입력하고 추론을 누르세요.
  2. 의미점수와 공통키워드를 보고 정답 뜻풀이를 좁혀가세요.
  3. 힌트는 단계적으로 열립니다.

정답은 서버에서만 알고 있고, 브라우저에는 내려오지 않습니다.
